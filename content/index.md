# 欢迎来到我的博客

这里是我的个人博客/数字花园，用来记录学习、项目与生活随笔。我更偏爱「把零散的想法串起来」，所以你会看到不少关联链接和阶段性整理。

## 从这里开始

- 先随便逛：看目录、点链接、顺着你感兴趣的主题一路走下去
- 用搜索：右上角搜索框（也可试试 `Ctrl+K` / `⌘K`）
- 用标签/图谱：如果你喜欢按主题探索，可以从标签页或图谱开始

## 我会写什么

- 技术实践：踩坑记录、工具链、自动化、工程化
- 学习笔记：书/课/文章的摘录与整理（尽量写出自己的理解）
- 项目复盘：做过什么、为什么这么做、哪里还能更好
- 生活随想：一些不成体系但值得记下的片段
