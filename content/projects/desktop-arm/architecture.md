> 系统架构图和技术选型说明

## 系统概览

```
┌─────────────────────────────────────────────────────┐
│                    上位机                           │
│         （Python / Web 控制界面）                   │
└─────────────────────┬───────────────────────────────┘
                      │ USB / WiFi
                      ▼
┌─────────────────────────────────────────────────────┐
│                   主控板                            │
│              （ESP32 / Arduino）                    │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐            │
│  │ 串口    │  │ PWM x6  │  │ ADC     │            │
│  └─────────┘  └─────────┘  └─────────┘            │
└───────┬─────────────┬─────────────┬─────────────────┘
        │             │             │
        ▼             ▼             ▼
┌───────────┐  ┌───────────┐  ┌───────────┐
│ 关节舵机   │  │ 末端执行器│  │ 位置反馈  │

└───────────┘  └───────────┘  └───────────┘
```

## 机械臂构型

| 关节 | 功能 | 运动范围 |
|------|------|----------|
| J1 | 底座旋转 | 0° ~ 180° |
| J2 | 肩关节 | 0° ~ 180° |
| J3 | 肘关节 | 0° ~ 180° |
| J4 | 腕关节俯仰 | 0° ~ 180° |
| J5 | 腕关节旋转 | 0° ~ 180° |
| J6 | 夹爪 | 开/合 |
