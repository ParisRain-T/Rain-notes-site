# 从这里开始

如果你是第一次来（或你是面试官/合作方），建议从这页快速了解本站的内容结构与我在做什么。

## 3 分钟了解我

- 我是谁：Rain（可替换成你的名字/昵称）
- 我做什么：`方向/岗位`（例如：后端 / 全栈 / 数据 / 嵌入式）
- 我在意什么：把问题讲清楚，把方案做扎实，把经验沉淀下来

## 你可能想看的

- [[about|关于我]]：更完整的介绍与经历
- [[now|近况]]：我最近在做什么（会持续更新）
- [[projects/index|Projects]]：项目总览（含 Demo/架构/亮点）
- [[posts/index|Posts]]：文章列表（相对成体系的长文）
- [[notes/index|Notes]]：随手记与知识卡片（更碎、更快）

## 怎么逛本站

- 直接搜索：右上角搜索框（也可试试 `Ctrl+K` / `⌘K`）
- 顺着链接：看到感兴趣的词就点进去，建立你自己的路径
- 看图谱/标签：如果你喜欢按主题探索，可以从图谱或标签页开始
