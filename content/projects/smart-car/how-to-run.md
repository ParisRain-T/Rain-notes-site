> 如何在本地运行项目（当前处于调研阶段，以下为预期流程）

## 前置条件

### 硬件

- [ ] 组装好的智能小车
- [ ] USB 数据线
- [ ] 电池（充满电）

### 软件

- [ ] Arduino IDE 或 PlatformIO
- [ ] 串口驱动（CH340 / CP2102）
- [ ] 串口调试工具（可选）

## 快速开始

### Step 1：克隆仓库

```bash
git clone https://github.com/ParisRain-T/smart-car.git
cd smart-car
```

### Step 2：配置开发环境

- 安装对应开发板的核心包（Arduino core / PlatformIO platform）
- 安装常用库（电机驱动、传感器）
- 选择正确的板型与串口

### Step 3：修改配置

- 根据接线修改引脚定义（PWM/方向/传感器）
- 如需联网功能，填入 WiFi 配置

### Step 4：编译烧录

- 编译固件并烧录到主控板
- 打开串口监视器确认启动日志

### Step 5：测试运行

- 先测试基础运动（前进/后退/转向）
- 再验证传感器读数与避障逻辑

## 常见问题

### Q: 电机不转？

可能原因：
1. 电源不足或电池电压过低
2. 电机驱动接线错误或使能脚未拉高
3. PWM 引脚与实际接线不一致

### Q: 传感器无数据？

可能原因：
1. VCC/GND 接线错误
2. 触发/回响引脚定义不一致
3. 采样间隔过短导致读数异常
