> 系统架构图和技术选型说明（调研阶段）

## 系统概览

```
┌─────────────────────────────────────────────────────┐
│                    上位机                           │
│         （Python / Web 控制界面）                   │
└─────────────────────┬───────────────────────────────┘
                      │ USB / WiFi
                      ▼
┌─────────────────────────────────────────────────────┐
│                   主控板                            │
│              （ESP32 / Arduino）                    │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐            │
│  │ 串口    │  │ PWM x6  │  │ ADC     │            │
│  └─────────┘  └─────────┘  └─────────┘            │
└───────┬─────────────┬─────────────┬─────────────────┘
        │             │             │
        ▼             ▼             ▼
┌───────────┐  ┌───────────┐  ┌───────────┐
│ 关节舵机   │  │ 末端执行器│  │ 位置反馈  │
│ x5-6      │  │ 夹爪/吸盘 │  │ 电位器    │
└───────────┘  └───────────┘  └───────────┘
```

## 机械臂构型

| 关节 | 功能 | 运动范围 |
|------|------|----------|
| J1 | 底座旋转 | 0° ~ 180° |
| J2 | 肩关节 | 0° ~ 180° |
| J3 | 肘关节 | 0° ~ 180° |
| J4 | 腕关节俯仰 | 0° ~ 180° |
| J5 | 腕关节旋转 | 0° ~ 180° |
| J6 | 夹爪 | 开/合 |

## 技术栈（候选方案）

| 层级 | 候选方案 | 选型标准 |
|------|----------|----------|
| 主控 | ESP32 / Arduino Mega / STM32 | IO/PWM 数量、生态、成本 |
| 舵机 | MG996R / MG90S / 数字舵机 | 扭矩、精度、成本 |
| 结构 | 3D 打印 / 亚克力 / 铝合金 | 强度、加工难度、成本 |
| 上位机 | Python + PyQt / Web | 开发效率、可视化 |
| 控制算法 | 正逆运动学 / 示教复现 | 实现复杂度、效果 |
