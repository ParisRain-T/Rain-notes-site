- MQTT / CoAP / HTTP
- LoRa / NB-IoT / BLE
- WiFi / Zigbee

### 硬件平台
- ESP32 / ESP8266
- STM32
- Raspberry Pi

### 云平台
- 阿里云 IoT
- AWS IoT Core
- 腾讯云 IoT

### 边缘计算
- Docker on ARM
- 边缘 AI 推理
